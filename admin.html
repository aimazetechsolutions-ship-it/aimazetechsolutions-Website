GitHub Auto-Publish Settings

When you click "Save & Publish", changes are committed directly to your GitHub repo and Cloudflare auto-deploys within 1–2 minutes.

How to get a GitHub Token:
1. Go to github.com/settings/tokens/new
2. Give it a name (e.g. "AimAze Admin")
3. Set expiration: No expiration
4. Check ✅ repo scope
5. Click "Generate token" and copy it here

🚀 Publishing to GitHub

Your changes will be committed to GitHub and Cloudflare will automatically redeploy your live website within 1–2 minutes.
