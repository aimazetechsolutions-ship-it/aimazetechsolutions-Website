VIDEO TECHNICAL REQUIREMENTS:
- Format: MP4
- Codec: H.264
- Recommended duration: 8 to 20 seconds loop
- Recommended size: below 5 MB each
- Resolution: 1920x1080 or 1280x720
- Keep videos muted; the website already autoplays muted/loop/playsinline.
- Avoid videos with large text in the footage because hero text appears on top.

HOW TO REPLACE:
1. Rename your selected video exactly as the relevant file name, e.g. home.mp4.
2. Put it inside assets/videos/.
3. Replace the existing file.
4. Deploy again.
5. Hard refresh browser: CTRL + SHIFT + R.

POSTER IMAGES:
Each video also has a poster image, e.g. home-poster.jpg.
You can replace poster files too. Poster image is shown while the video loads.

CONTENT.JSON MAPPING:
The page videos are controlled in content.json under "pageVideos".
Example:
"home": { "mp4": "assets/videos/home.mp4", "poster": "assets/videos/home-poster.jpg" }

Do not paste YouTube iframe code in the MP4 field. For premium setup, use MP4 file paths only.
